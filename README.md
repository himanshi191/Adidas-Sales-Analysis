# Adidas-Sales-Analysis
📊 Adidas Sales Analysis Dashboard
📌 Project Overview

The Adidas Sales Analysis Dashboard is a data analytics project designed to analyze and visualize Adidas sales performance across different regions, products, and time periods.
The dashboard provides insights into revenue trends, product performance, sales distribution, and regional performance, helping businesses make better data-driven decisions.

This project demonstrates the use of data analytics, data visualization, and business intelligence tools to transform raw sales data into meaningful insights.

🎯 Objectives

The main objectives of this project are:

Analyze Adidas sales performance

Identify top-performing products

Understand regional sales distribution

Track revenue generation trends

Discover patterns in customer purchasing behavior

Provide actionable insights using visual dashboards

📂 Dataset Description

The dataset contains information related to Adidas product sales.

Key Columns in Dataset

Retailer – Name of the retailer selling Adidas products

Region – Geographic region of sales

State – State where sales occurred

City – City where products were sold

Product – Type of Adidas product

Price per Unit – Price of a single unit

Units Sold – Number of units sold

Total Sales – Revenue generated

Operating Profit – Profit from sales

Sales Method – Online / In-store sales

📊 Dashboard Features

The dashboard provides multiple analytical views:

1️⃣ Sales Overview

Total Revenue

Total Units Sold

Total Profit

Overall sales performance

2️⃣ Region-wise Sales Analysis

Revenue distribution across different regions

Identification of highest revenue generating regions

3️⃣ Product Performance

Best-selling Adidas products

Revenue generated by each product

4️⃣ City & State Analysis

Sales contribution by cities and states

Market performance across locations

5️⃣ Sales Method Analysis

Comparison between online sales vs in-store sales

📈 Key Insights

Some important insights derived from the dashboard:

Certain regions contribute significantly to overall revenue.

Some Adidas products dominate the market in terms of units sold.

Online sales channels are becoming increasingly important.

Sales patterns vary significantly across different cities and states.
